Create a folder with your store name inside of your themekit folder. Eg: `the-man-company` 

Run `npm init`.
Add devDependencies and scripts from this file. https://gitlab.com/-/snippets/1955631
Add gitignore if required - https://gitlab.com/-/snippets/1985641
Add webpack.config.js - https://gitlab.com/-/snippets/1954750
Create two folders: (i) javascript (ii)scss
Create app.js in javascript folder and app.scss in scss folder.
Create a folder name theme and add config.yml - https://gitlab.com/-/snippets/1972255
Run `npm install`  from root directory. 
Run npm run download and run run start.
